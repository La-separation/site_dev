Historique des mises à jour | Application en ligne | Application pour tablette



  • 2.1.x
    • 2.1.2 - Passage des fichiers audio au format .ogg
      • Passage du format .wav en .ogg : Gain sur la taille de l'application (14Mo -> 3Mo).
    • 2.1.1 - Animations dynamiques pour les mots
      • Animation simple "couper" dynamique.
      • Disponible vers la gauche ou vers la droite.
      • Pas de gestion des évènements.
    • 2.1.0 - Commencement du récit
      • Affichage dynamique des mots.
      • Séparation de l'écran en lignes pour l'affichage.
      • Uniquement pour la police coupable_haut.
  • 2.0.x
    • 2.0.5 - Résolution du bug audio sur tablette
      • Le son fonctionne pour toutes les plate-formes (PC, tablette).
    • 2.0.4 - Restructuration du code
      • Division des scripts JS en plusieurs petits scripts pour faciliter grandement le développement.
    • 2.0.3 - Modification du logo de l'introduction
      • Amélioration graphique du logo, plus circulaire et plus symétrique.
    • 2.0.2 - Ajout de la version PC
      • Gestion centralisée des evènements peu importe le support.
      • Bug audio : le son n'est disponible que sur PC.
      • Détection manuelle pour savoir si on est sur tablette ou sur PC.
    • 2.0.1 - Gestion du multi-plateforme mobile
      • Passage de Cordova à Phonegap.
      • On peut désormais compiler l'application dans le cloud (sur des serveurs d'Adobe), pour toutes les plateformes d'un coup.
      • On obtient un "lien de partage" permettant de télécharger l'application depuis n'importe quel tablette (en allant sur le lien avec une tablette ou en scannant le QRcode).
    • 2.0.0 - Code source initial
